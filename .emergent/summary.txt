- **Two Modes:**
    1.  **Bulk Mode:** A single shared design for a group, with quantity controls for different sizes and an option for a list of names for back printing.
    2.  **Multi-Design Mode:** Different designs for each person within a single order.
- **Core Features:**
    - A gallery of pre-made t-shirt design templates (e.g., Stag, Hen).
    - Customers can upload a photo, from which the app should automatically extract the head.
    - A canvas editor where users can position, scale, and rotate the extracted head onto the template.
    - Users can edit text fields on the design.
    - An optional add-on for printing a name and number on the back.
- **Technical & Integration:**
    - The builder must integrate with the Shopify cart, adding configured items with all custom details as line item properties.
    - The final output must include print-ready PNG files (front and back) and the original uploaded photo for the store owner.
    - These assets must be stored persistently (e.g., S3, Cloudinary) and be downloadable from a simple admin dashboard.
- **MVP Goal:** A functional flow from template gallery -> builder -> add to cart -> Shopify checkout -> admin download page.
